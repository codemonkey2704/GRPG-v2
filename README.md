# gRPG v2
PHP, MySQL, HTML, CSS.
Text-based online RPG

### How do I get set up? ###

* Upload the files to your public_html (or htdocs/www)
* Create a MySQL database and user
* Go to the URL of wherever you just uploaded them - go through it and fill in the details
* Login

### I found a bug! ###
